Guía de instalación del EA
1. Descarga FirebaseBridge.mq5 y cópialo a MQL5/Experts/
2. Compílalo en MetaEditor (F7)
3. En MT5: Herramientas → Opciones → Asesores Expertos
   ✓ Activar "Permitir solicitudes web"
   ✓ Agregar URL: https://firestore.googleapis.com
4. Arrastra el EA a cualquier gráfico (ej. EURUSD M1)
5. Configura los parámetros:
   • FirebaseProject = tu proyecto ID
   • FirebaseKey = tu API key
   • Room = ---
6. ¡Listo! Cada trade aparecerá automáticamente en la bitácora.
